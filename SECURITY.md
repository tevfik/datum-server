# Security Policy

## Supported Versions

Currently, only the latest release of Datum Server is actively supported with security updates. We recommend keeping your deployment up to date with the latest tag.

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Security is a core priority for Datum Server. If you discover a vulnerability or security issue, please **DO NOT** create a public GitHub Issue.

Instead, please report it privately:

1. Send an email to the repository owner or open a private advisory draft via GitHub Security Advisories.
2. Include steps to reproduce the issue.
3. Describe the potential impact and any known workarounds.

We will acknowledge your report as soon as possible and work with you to plan a fix. Once resolved, we will publish a security patch.
