# Datum Server Security Reports

This directory contains automatically generated security reports for the backend, tracking dependencies and known vulnerabilities.

## Files
- `sbom.cdx.json`: The Software Bill of Materials (SBOM) in CycloneDX 1.5 JSON format. It lists all direct and transitive dependencies along with their license metadata.
- `govulncheck.txt`: The latest vulnerability scan report generated by Go's official `govulncheck` tool. It analyzes the compiled call graph to see if any known CVEs are actually reachable by our code.

## Generating Reports

To regenerate these files after bumping dependencies in `go.mod`, run the included script:

```bash
cd backend
./security/generate.sh
```

### Requirements
- **Go toolchain** (`go` command) must be installed.
- The script will automatically install `cyclonedx-gomod` and `govulncheck` if they are not already installed on your system.
