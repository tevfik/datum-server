# EIF Pre-trained Models

This directory contains pre-trained model bundles for EIF demos.

## Available Models

| Model | Size | Demo | Description |
|-------|------|------|-------------|
| `mnist_cnn.eif` | ~50KB | mnist_cnn | 3-layer CNN for digit recognition |
| `kws_model.eif` | ~30KB | kws_demo | Keyword Spotting (wake words) |
| `gesture_nn.eif` | ~10KB | gesture_recognition | 6-gesture classifier |
| `anomaly_iforest.eif` | ~5KB | anomaly_detection | Isolation Forest for anomaly detection |

## Model Format

Models use the EIF binary format with:
- Header: Magic bytes, version, layer count
- Layer metadata: Type, dimensions, activation
- Weight data: Float32 little-endian

## Usage

```c
#include "eif_neural.h"

// Load model from file
eif_model_t model;
eif_neural_load(&model, "models/mnist_cnn.eif", &pool);

// Run inference
float output[10];
eif_neural_invoke(&model, input, output);
```

## Generating Models

Models are generated by training scripts in `tools/`:

```bash
# Generate all models (requires Python + NumPy)
python3 tools/generate_models.py

# Generate specific model
python3 tools/generate_models.py --model mnist_cnn
```
